# More higher order functions

Begin:

```racket
(define a
  (begin ; for things that need to go in order and have side effects
    (+ 2 3)
    (* 4 5)
    (/ 12 3)))
; a
; > 4
```

Input/output:

```racket
(display "Hello")
; Hello
(newline)
; \n
(printf "The string is ~a~n" "hello") ; not tested on this
; The string is hello
```

Multiple airity functions:

```racket
(define f
  (lambda x
    (map (lambda (y) (+ y 1)) x)))
(f 1 2 3)
; '(2 3 4)
```
