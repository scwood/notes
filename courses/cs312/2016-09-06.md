# Modular exponentiation

```
modexp(x, y, n)        ----> computes x^y mod N
  if y == 0 return 1
  z = modexp(x, floor(y / 2), n)            ----> z is a long
  if y is even
    return z^2 mod n
  else
    return x * (z^2 mod n) mod n
```
